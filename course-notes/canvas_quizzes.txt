3. What does the following code do on a linked list? ('head' points to the head of the linked list)
      node* prev = NULL;
      node* current = head;
      node* next;
      while (current != NULL)
      {
      next = current->next; 
      current->next = prev; 
      prev = current;
      current = next;
      }
      head = prev;
Answer: Reverse a Linked List
Explanation:

1. What does the following pseudocode function do?
      func(Queue Q)
      {
          Stack S;
          while (!Q.isEmpty())
          {
          S.push(Q.pop());
          }
          while (!S.isEmpty())
          {
          Q.push(S.pop());
          }
      }
Answer: Reverses the Queue
